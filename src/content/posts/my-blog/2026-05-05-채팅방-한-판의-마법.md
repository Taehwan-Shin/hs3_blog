# 채팅방 한 판의 마법: AI 에이전트와 함께한 콘텐츠 자동화의 오늘

안녕하세요, 백제입니다. 🏯

오늘은 **하루의 대화를 한 편의 블로그 포스팅으로 자동 게시**하는 과정을 직접 기록해 봅니다. 단순한 기술 소개가 아니라, 실제로 *지금 이 순간에도 작동하고 있는* 시스템의 생생한 기록입니다.

---

## 🎯 오늘의 출발점: 하나의 간단한 요청

우리 채팅방(고교 삼국시대)에서 누군가 이렇게 말했습니다.

> *"이 채팅방에서 나눈 대화를 정리해서 블로그에 게시해줘."*

**단 한 줄.** 하지만 이 한 줄이 시스템 전체를 움직였습니다.

---

## 🔧 자동화가 작동하는 5단계

![시스템 워크플로우](attachments/hermes-hero.png)

### 1단계: 파일 감지 & 구조 파악

먼저 시스템은 다음과 같은 명령으로 작업 환경을 확인했습니다.

```bash
# 가이드 파일 확인
/workspace/Brain/GUIDE_FOR_HERMES.md

# 블로그 저장소 확인
/workspace/vault-blog/

# 동기화 스크립트 확인
/workspace/vault-blog/sync.sh
```

여기서 중요한 것은 **GUIDE_FOR_HERMES.md**라는 운영 매뉴얼이 존재한다는 점입니다. 이 파일이 시스템의 DNA 역할을 합니다.

### 2단계: 콘텐츠 변환 (GUIDE 기준)

GUIDE에서 정의한 프론트매터 스탠다드에 맞춰 게시글을 작성합니다.

```yaml
---
title: "채팅방 한 판의 마법..."
description: "고교 국사 선생님과 Hermes AI..."
date: 2026-05-05
author: baekje     # 백제 필진: AI 기술의 깊이 있는 이해
image: "attachments/hermes-hero.png"
tags: ["AI에이전트", "블로그자동화"]
draft: false
---
```

**백제 필진의 관점:** 여기서 강조하는 것은 *기술적 깊이*입니다. "프롬프트 한 줄로 끝까지 간다"는 것이 백제 필진이 추구하는 AI 기술의 본질입니다.

### 3단계: 저장소 동기화 (sync.sh)

```bash
cd /workspace/vault-blog
./sync.sh
```

이 스크립트가 `Brain/200 blog/` 폴더 전체를 `vault-blog/src/content/posts/my-blog/`로 복사합니다.

### 4단계: 이미지 최적화 (sync-images.js)

```bash
node scripts/sync-images.js
```

PNG/JPEG 이미지를 WebP로 자동 변환하여 퍼포먼스를 높입니다. Astro 빌드 엔진이 자동으로 서빙합니다.

### 5단계: Git 커밋 & 푸시 (Vercel 자동 배포)

```bash
cd /workspace/vault-blog
git add .
git commit -m "Add new post: 채팅방 한 판의 마법..."
git push origin master
```

**핵심 조건:** 커밋 작성이 반드시 `gssth8286@gmail.com`이어야 합니다. 다르면 Vercel이 배포를 거부합니다.

## 💡 이 시스템이 주는 교훈

### 교사의 시각에서본 AI의 가치

> "수업 준비하랴, 연구하랴, 집필하랴. 교사에게 시간은 가장 귀한 자원입니다."

오늘 확인한 이 자동화 파이프라인은 단순한 '편리한 도구'가 아닙니다. **교사의 시간을 되찾는 인프라**입니다.

| 작업 | 수동 | 자동화 |
|------|------|--------|
| 파일 읽기 | 5-10분 | 실시간 |
| Markdown 변환 | 15-20분 | 자동 |
| 이미지 처리 | 10-15분 | 자동 (WebP) |
| 배포 | Git 명령 + Vercel 대기 | 자동 푸시 |
| **합계** | **35-45분** | **실시간** |

### 프롬프트가 곧 명령어

오늘의 핵심 인사이트:

> **"사용자가 management할 수 있는 것은 오직 요청(프롬프트)뿐입니다."**

시스템은 이미 설계되어 있습니다. 구조는 완성되었습니다. 우리가 해야 할 일은 정확한 언어로 요청하는 것.
